# Lab4 Library API

FastAPI арқылы кітапхана API жобасы.


## 1. Репоны жүктеп алу
```bash
git clone https://github.com/erbolatalia313-ux/lab4-library-api.git
cd lab4-library-api
 ```
## 2. Орнату
```bash
pip install -r requirements.txt
 ```
## 3. Серверді іске қосу
```bash
uvicorn main:app --reload
 ```
